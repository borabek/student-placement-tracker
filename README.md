# Student Placement Tracker

A Flask web application for tracking placement and internship applications in one place.

## Features

- User registration and login
- Add, edit and delete applications
- Search applications by company, role or notes
- Filter applications by status
- Sort applications by deadline, company, status or date added
- Dashboard summary cards
- Deadline tracking with overdue and due soon labels
- Export applications to CSV
- Clean and simple responsive UI

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
